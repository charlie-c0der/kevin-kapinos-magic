# Kevin Kapinos - Mentalist Website

A sophisticated, dark-themed website for Chicago mentalist and psychological entertainer Kevin Kapinos. Built with Vue 3, TypeScript, and advanced GSAP animations.

## 🎭 About the Project

This website showcases Kevin Kapinos's unique approach to mentalism and psychological entertainment. Unlike traditional magic websites, this focuses on the psychology and science behind mentalism, creating an atmosphere of mystery and intellectual intrigue.

### 🎨 Design Philosophy
- **Dark, mysterious aesthetic** with deep blues and purples
- **Psychology-focused branding** emphasizing the mind and human behavior
- **Sophisticated animations** using GSAP for professional appeal
- **Scientific credibility** combined with entertainment value

### 🧠 Differentiation from Traditional Magic Sites
- Emphasizes **psychology over illusion**
- Features **educational content** about mentalism
- **Professional, corporate-friendly** presentation
- **Mysterious but not supernatural** - grounded in real techniques

## 🌟 Key Features

### **Comprehensive Content**
- **Home**: Hero section with mind-reading focus, performance types, testimonials
- **About**: Professional biography, timeline, expertise areas, credentials

### **Technical Excellence**

## 🔧 Development

### **Prerequisites**
- Node.js 18+ 
- npm or yarn package manager

### **Installation**
```bash
# Clone the repository
git clone [repository-url]
cd kevin-kapinos-magic

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### **Development Scripts**
```bash
npm run dev          # Development server
npm run build        # Production build
npm run preview      # Preview production build
npm run type-check   # TypeScript type checking
npm run lint         # ESLint code analysis
```
